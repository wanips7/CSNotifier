[EN]

A program that notifies about the start of a new round of the Counter-Strike 2 game when the game window is hidden.

This program uses only Game State Integration, without reading game memory.

The icon was generated on [alice](https://alice.yandex.ru)

How to use:

```
 1. Click "Copy GSI file" button
 2. Click "Copy command" button
 3. Paste the copied text into the game launch options
 4. Click "Start" button
```

[Download](https://github.com/wanips7/CSNotifier/releases/latest)

[RU]

Программа, которая уведомляет о начале нового раунда игры Counter-Strike 2, когда окно игры свернуто.

Эта программа использует только интеграцию состояния игры, без чтения памяти игры.

Иконка сгенерирована на [alice](https://alice.yandex.ru)

Как использовать:

```
 1. Нажмите кнопку "Copy GSI file" 
 2. Нажмите кнопку "Copy command"
 3. Вставьте скопированный текст в параметры запуска игры.
 4. Нажмите кнопку "Start"
```

[Скачать](https://github.com/wanips7/CSNotifier/releases/latest)
